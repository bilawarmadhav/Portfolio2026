# Madhav Bilawar Portfolio

A highly modular, premium, and performant vanilla JavaScript portfolio built with Vite.

## Features
- **Vanilla JS Modules**: Codebase split into logical chunks.
- **CSS Architecture**: Split into variables, resets, specific component modules.
- **Dynamic Data**: Populated using JSON files for easier maintenance.
- **Animations**: Uses Intersection Observer for scroll animations and CSS keyframes for floating elements.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
